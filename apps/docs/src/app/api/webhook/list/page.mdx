# Webhook 목록 조회

### 개요

현재 API Key 소유자가 등록한 Webhook 목록을 조회합니다. 보안상 `secret`은 응답에 포함되지 않습니다.

### 인증

```http
X-API-KEY: your-api-key-here
```

### 엔드포인트

```http
GET /v1/webhooks
```

### 권한 범위

`WEBHOOK_WRITE` 권한 범위가 필요합니다.

### 요청 파라미터

없습니다.

### 응답 형식

| 필드         | 타입                    | 설명         |
|------------|-----------------------|------------|
| `webhooks` | `List<WebhookResDto>` | Webhook 목록 |

#### Webhook 객체 (WebhookResDto)

| 필드           | 타입                  | 설명                  | 예시                                       |
|--------------|---------------------|---------------------|------------------------------------------|
| `id`         | `Long`              | Webhook ID          | `1`                                      |
| `target_url` | `String`            | 수신 URL              | `https://example.com/webhooks/datagsm`   |
| `events`     | `Set<WebhookEvent>` | 구독 이벤트 목록           | `["STUDENT_GRADUATED", "CLUB_CREATED"]` |
| `is_active`  | `Boolean`           | 활성화 여부              | `true`                                   |
| `created_at` | `String`            | 생성 일시 (`ISO-8601`) | `2026-05-13T10:00:00`                    |

> `secret` 필드는 보안상 목록 조회 응답에 포함되지 않습니다. 등록 시 반환된 값을 별도로 저장하여 사용하세요.

### 요청 예시

```bash
curl -X GET "https://openapi.datagsm.kr/v1/webhooks" \
  -H "X-API-KEY: your-api-key-here"
```

### 응답 예시

```json
{
  "webhooks": [
    {
      "id": 1,
      "target_url": "https://example.com/webhooks/datagsm",
      "events": ["STUDENT_GRADUATED", "CLUB_CREATED"],
      "is_active": true,
      "created_at": "2026-05-13T10:00:00"
    },
    {
      "id": 2,
      "target_url": "https://another.example.com/hook",
      "events": ["PROJECT_CREATED", "PROJECT_UPDATED", "PROJECT_DELETED"],
      "is_active": true,
      "created_at": "2026-05-13T11:30:00"
    }
  ]
}
```

### 오류 응답

| 상태 코드              | 설명                      |
|--------------------|-------------------------|
| `401 Unauthorized` | API 키가 유효하지 않거나 만료됨     |
| `403 Forbidden`    | `WEBHOOK_WRITE` 권한 범위 부족 |

public class WebhookAPI {

        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://openapi.datagsm.kr/v1/webhooks"))
